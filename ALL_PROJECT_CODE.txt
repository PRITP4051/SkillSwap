DATA FLOW

1. User SignUp
   HTML Form → JavaScript registerUser() → Save to localStorage → Redirect

2. User Login
   HTML Form → JavaScript loginUser() → Set currentUser → Navigate

3. Create Listing
   HTML Form → JavaScript createListing() → Add to listings array → Save

4. Search Skills
   Input → searchListings() → Filter listings → Re-render HTML

5. Send Message
   Form Input → sendMessage() → Add to messages → Update UI

6. Leave Review
   Form → createReview() → Add to reviews → Update teacher rating

================================================================================
                        TROUBLESHOOTING
================================================================================

COMMON ISSUES & SOLUTIONS

Issue: Blank page
- Check browser console for errors
- Verify all file paths are correct
- Ensure folder structure maintained
- Try different browser

Issue: Login fails
- Verify signup was completed
- Check password is correct
- Check email is lowercase
- Check localStorage isn't full

Issue: Listings not showing
- Create a new listing first
- Check browser localStorage
- Clear browser cache
- Verify js/app.js loaded

Issue: XML parser errors
- Verify XML file path correct
- Check XML syntax validity
- Try xml-test.html
- Check browser console

Issue: Styles not loading
- Verify css/styles.css path
- Check file wasn't moved
- Clear browser cache
- Hard refresh (Ctrl+Shift+R)

Issue: Messages not saving
- Check localStorage available
- Verify recipients exist
- Check message content not empty
- Clear localStorage if full
