================================================================================
                            QUICK START GUIDE
================================================================================

1. OPEN IN BROWSER
   - Open index.html in any modern web browser
   - No build process needed
   - No server required (runs entirely client-side)

2. SIGN UP
   - Click "Get Started" or go to signup.html
   - Create account with name, email, password
   - Auto-login after registration

3. EXPLORE
   - Browse available skills
   - Search for specific skills
   - View teacher profiles and ratings

4. BOOK SKILLS
   - Click "Book" on any skill
   - Booking added to dashboard
   - Teacher sees request

5. CREATE LISTING
   - Go to Dashboard
   - Click "Create Listing"
   - Enter title, description, price
   - Start earning!

6. TEST XML
   - Open xml-test.html
   - Click "Parse XML File"
   - Verify all data loads
   - Check each tab
